## 项目简介 Project Brief
自己在学习机器学习和深度学习中的一些算法和例子  
一些算法有自己手写，一些直接用sklearn包进行测试；有些算法我有写算法原理详解的文章，详细见下方算法列表  
个人博客地址JYRoy https://www.cnblogs.com/jyroy

1. Python 3.7.1
2. Pycharm 2017.1 PROFESSIONAL
